Question 1

Trình bày các cách để tích hợp Bootstrap vào một trang HTML/CSS bình thường trong điều kiện đang có kết nối Internet? Trong Bootstrap đã được biên dịch gồm có những gì?


Có 2 cách để tích hợp Bootstrap vào trong file html/css trong điều kiện có kết nối internet:

Cách 1:
+ Tải file bootstrap vào trong cùng folder với file html/css và khai báo đường link tại <header> của file html:
<header>
...
<link rel="stylesheet" href="Name-file-boostrap/css/bootstrap.min.css">
...
</header>
Cách 2:
+ Dẫn trực tiếp đường link của file bootstrap trên internet (tuy nhiên sẽ gây chậm trong điều kiện tốc độ mạng kém):
<header>
...
<link rel="stylesheet" href="//netdna.bootstrapcdn.com/bootstrap/3.1.1/css/bootstrap.min.css" >
...
</header>
Trong Bootstrap đã được biên dịch gồm:
                                  bootstrap/
                                  ├── css/
                                  │   ├── bootstrap.css
                                  │   ├── bootstrap.min.css
                                  │   ├── bootstrap-theme.css
                                  │   └── bootstrap-theme.min.css
                                  ├── js/
                                  │   ├── bootstrap.js
                                  │   └── bootstrap.min.js
                                  └── fonts/
                                      ├── glyphicons-halflings-regular.eot
                                      ├── glyphicons-halflings-regular.svg
                                      ├── glyphicons-halflings-regular.ttf
                                      └── glyphicons-halflings-regular.woff

Question 2

Responsive là gì? Làm thế nào nếu muốn vô hiệu hóa tính năng Responsive khi đang sử dụng Bootstrap?

Responsive là gì?

Hiểu một cách đơn giản Web Responsive là phong cách thiết kế website làm sao cho phù hợp trên tất cả các thiết bị, mọi độ phân giải màn hình. Một website đẹp trên máy tính nhưng khi xem trên điện thoại bố cục lộn xộn, hình ảnh, bài viết không cân đối thì không thể gọi là Web Responsive
Ví dụ về Web Responsive:
Bạn mở trang chủ website Responsive, rồi thu nhỏ trình duyệt hoặc đổi độ phân giải màn hình, bạn sẽ thấy website của mình không hề xuất hiện Scrollbar ngang, mà nó sẽ tự động co dãn sao cho phù hợp với chiều rộng màn hình máy tính bạn.

Vô hiệu hóa tính năng Responsive:
  1. Hãy bỏ thẻ <meta> viewport đã được đề cập đến trong tài liệu CSS
  2. Hãy ghi đè thuộc tính width trên .container đối với mỗi cột trong hệ thống lưới, ví dụ width: 970px !important; Hãy chắc chắn rằng, các mã CSS tùy biến này được tích hợp vào sau CSS mặc định của Bootstrap. Bạn cũng có thể tránh sử dụng !important bằng các media query hoặc một số selector-fu.
  3. Nếu bạn sử dụng thanh điều hướng, hãy vô hiệu hóa toàn bộ các tính năng thu gọn và mở rộng trên thanh điều hướng đó.
  4. Đối với các bố cục dạng lưới, hãy sử dụng class .col-xs-* bằng cách thêm vào hoặc thay thế cho các class medium/large. Đừng lo lắng, hệ thống lưới trên các thiết bị có màn hình siêu nhỏ sẽ tự động co giãn đối với mọi độ phân giải.

Tìm và cho ví dụ về cách để thực hiện các yêu cầu sau:
+ Chèn 1 button 1 biểu tượng trong Button đó.
+ Chèn 1 thanh Progress bar (tiến trình) tương tự trong CV ở Task 1.2.
+ Chèn các loại cảnh báo (Alert) trong trang như Succes, Info, Danger, Warning.
+ Chèn 1 menu (Navigation) trong đó có 1 mục có menu con xổ xuống.



Chèn 1 button 1 biểu tượng trong Button đó:

Đây là ví dụ một button có biểu tượng ngôi sao trong github:

Cách thực hiện:

- Bước 1: Tạo button với bootstrap
<button class="btn btn-outline-primary"></button>

-Bước 2: Dùng Fontawesome để vào 1 icon trong button
<button class="btn btn-outline-primary"> <i class="fas fa-star"></i> Star </button>
- Kết quả

Chèn 1 thanh Progress bar (tiến trình) tương tự trong CV ở Task 1.2:

Cách thực hiện:
<div class="progress">
<div class="progress-bar" role="progressbar" style="width: 25%;" aria-valuenow="25" aria-valuemin="0" aria-valuemax="100">25%</div>
</div> - Kết quả

25%

Chèn các loại cảnh báo (Alert) trong trang như Success, Info, Danger, Warning:

Cách thực hiện:
<div class="alert alert-warning">Đây là cảnh báo nguy hiểm</div>

Ngoài ra còn một số loại alert khác:
+ warning : cảnh báo
+ primary : thông thường
+ secondary : thông báo thứ cấp
+ success : thông báo thành công
+ danger: thông báo nguy hiểm
- Kết quả

Chèn 1 menu (Navigation) trong đó có 1 mục có menu con xổ xuống:

  • Active
  • Dropdown
    Action Another action Something else here

  • Disabled
