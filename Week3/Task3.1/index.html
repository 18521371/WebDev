Question 1

Những vị trí nào có thể đặt Javascript trong file HTML và dặt theo cú pháp nào? Nên đặt Javascript tại vị trí nào để có hiệu suất tốt nhất? Cho ví dụ minh họa.

Những vị trí có thể đặt Javascript trong file HTML:

Vị trí 1:
+ Đặt trong phần <head> của file html
Cú pháp:
<head>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
Vị trí 2:
+ Đặt trong phần <body> của file html:
Cú pháp:
<body>
...
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</body>
Vị trí 3:
+ Đặt ở trong file.js bên ngoài file html:
Cú pháp:
Tạo file myScript.js bên ngoài file html rồi dẫn vào theo cú pháp bên dưới:
<body>
...
<script src="myScript.js">
</script>
</body>
Nên đặt Javascript tại vị trí bên ngoài để có hiệu suất tốt nhất:

Bởi vì:
+ Dễ phân biệt giữa html và javascript code
+ Làm cho html và javascript dễ đọc và bảo trì
+ Để file js bên ngoài có thể tăng tốc độ tải trang

Question 2

Trong Javascript có những kiểu dữ liệu nào? Làm thế nào để tạo một biến để lưu trữ giá trị tương ứng với những kiểu dữ liệu đó?

Trong Javascript có những kiểu dữ liệu:

+ Kiểu dữ liệu số học
+ Kiểu dữ liệu chuỗi (string)
+ Kiểu dữ liệu boolean
+ Kiểu dữ liệu mảng array
+ Kiểu dữ liệu đối tượng object
+ Kiểu dữ liệu undefined

Để tạo một biến để lưu trữ giá trị tương ứng với những kiểu dữ liệu trên:
Ví dụ cụ thể:
Kiểu dữ liệu số học:
<script>
  var a = 22;
  var b = -1993;
</script>
Kiểu dữ liệu string:
<script>
 var a = "Tài liệu học HTML";
  var b = 'Lập Trình Web';
</script>
Kiểu dữ liệu boolean:
<script>
 var a = true;
  var b = false;
  var c = 6 > 2;
  var d = 6 > 10;
</script>
Kiểu dữ liệu undefined:
<script>
  var myName; //biến myName sẽ có giá trị là undefined và kiểu dữ liệu là undefined
</script>


<script>
  var myName = "Nguyễn Thành Nhân"; //biến myName có kiểu dữ liệu là string
  myName = undefined; //biến myName có kiểu dữ liệu là undefined
</script>
Kiểu dữ liệu object:
<script>
 var SinhVien = {
   name:"Nhân",
   gender:"Nam",
   year:1993
}
</script>
Kiểu dữ liệu array:
<script>
   var mobile = ["HTC","Nokia","SamSung"];
</script>

Question 3

Làm thế nào để truy xuất (lấy giá trị) trong 1 thẻ HTML bất kỳ và gán vào 1 biến để trong Javascript theo ID, theo Class, theo Name, theo tên thẻ HTML và theo CSS Selector (ví dụ .class, #id)? Cho ví dụ minh họa.

Theo ID:
ví dụ:
var element = document.getElementById('idname');
Theo Class:
var element = document.getElementByClassName('ClassName');
Theo thẻ html:
var element = document.getElementByTagName('ClassName');
Theo CSS selector :
var element = document.querySelectorAll("selector.css");
