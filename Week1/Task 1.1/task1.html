Câu hỏi

  1. So sánh file HTML có định dạng .html và .htm. Nếu không có các thẻ , và <body> thì có ảnh hưởng gì trong file HTML không?
  2. Có những cách nào để hiển thị các thẻ HTML để hiển thị ra bên ngoài mà không bị HTML xử lý. Ví dụ hiển thị các nội dung như </br> <div><h1>Hello HTML</h1></div>
  3. Làm thế nào để tạo vùng chọn để định dạng trong CSS cho một tag, một class, một ID?
  4. Tìm và cho ví dụ về các thẻ có các chức năng sau:
    + Gạch chân, kẻ đường ngang, chỉ số trên/dưới.
    + Tạo bảng (có ghép dòng, ghép cột).
    + 4 hiệu ứng chữ chạy từ 4 hướng (trái, phải, trên, dưới).

Trả lời:

Câu a | Câu b | Câu c | Câu d
Câu a)
- html và .htm đều là phần mở rộng của file html tuy nhiên không có sự khác biệt là mấy bởi vì trước kia có nhiều hệ hành khác nhau nên sinh ra các file mở rộng khác nhau. Có 1 điểm khác biệt là nếu 2 file có phần mở rộng .html và .htm thì máy tính sẽ ưu tiên mở file .htm
- Nếu không có các thẻ , và <body> thì vẫn không ảnh hưởng gì đến file html
Câu b)
Có 2 cách để display các thẻ html mà không bị html nhầm lẫn:
+ Dùng Character Entity Name: &lt; &gt;
+ Dùng Character Entity Number: &#38 &#38
Câu c)
- Chọn theo tên tag:
a {
color : red;
}
- Chọn theo class:
.class {
color : red;
}
- Chọn theo ID:
#ID {
color : red;
}
